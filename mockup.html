Clovertecno

📅 Mar 2025 — Ago 2025 👥 Colaborador 🏢 Cliente: Clovertecno
Ver sitio live ↗ Ver repositorio ↗

Resumen

E-commerce de productos tecnológicos con sistema de compras, integración a Mercado Pago y módulo de gestión de productos para el dueño del negocio.

Mi rol

Participé como desarrollador colaborando en el módulo de gestión de productos (alta, edición, stock) y en la integración del checkout con Mercado Pago. Trabajé con el equipo en optimización de consultas MySQL para el catálogo y en componentes front-end del flujo de compra.

Stack técnico

PHP 8 Laravel MySQL MercadoPago SDK Bootstrap jQuery

Galería

[ home con catálogo ]
[ producto detalle ]
[ checkout MP ]
[ panel admin ]

Desafíos resueltos

  • Integración del webhook de Mercado Pago: manejo de estados (pendiente / aprobado / rechazado) y actualización de stock atómica.
  • Optimización de queries del listado de productos con filtros múltiples — reducción de tiempo de carga.
  • Panel de administración con CRUD completo y gestión de imágenes de productos.
